# 健身中心小幫手


## 主要功能：
* 查詢公有健身中心即時人流資訊(以北部為主)
* 輸入地標返回最近三間運動中心資訊

## 系統建置：
* python 3.6
* Dialogflow 理解語意
* OpenData Source : [全國運動場館資訊網](https://iplay.sa.gov.tw/webapi)
